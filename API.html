Création d'une API REST d'une pizzeria

Moi-même et mon camarade de classe Sacha Debusschère avons créé pour l'un de nos projets à l'université une API REST.

Cette API REST est celle d'une pizzeria fictive.

Nous avons appris à gérer les erreurs web, faire une API cohérente et fonctionnelle et utiliser efficacement JavaEE.
